# Incident Response Plan

Date: 2026-02-15
Owner: Security Officer (TBD)

## Triggers
- Unauthorized access attempts
- Bulk data access anomalies
- Webhook signature failures

## Response Steps
1. Detect and log the incident
2. Triage severity
3. Contain affected systems
4. Notify stakeholders
5. Post-incident review

## Reporting Timelines
- Internal report: within 24 hours
- Regulator: per NDPR requirements
